# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI.
The main objective of the project is to load and analyze a dataset, clean and transform data, perform SQL analysis, build interactive dashboards, and generate business insights.

The project covers:

* Data Loading using Python
* Exploratory Data Analysis (EDA)
* Data Cleaning & Transformation
* SQL Query Analysis
* Power BI Dashboard Creation
* Report Generation
* Presentation Creation using Gamma

## How to Run the Project

### Step 1: Clone the Repository

```bash
git clone <repository-link>
```

### Step 2: Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Step 3: Open Jupyter Notebook

```bash
jupyter notebook
```

### Step 4: Run SQL Queries

* Import dataset into PostgreSQL/MySQL/SQL Server
* Execute SQL scripts from the `sql_queries` folder

### Step 5: Open Power BI Dashboard

* Open `.pbix` file in Power BI Desktop
